# Proyecto-Eban

En este proyrcto tengo planeado hace un juego tipo RPG mundo abierto enfocado en la exploracion del mundo, donde no habra una historia fija. se podran explorar 5 mundos siendo empezando siempre en el bosque donde en un inicio habra un pqueño tutorial para entender el movimiento y las interacciones con el entorno. depues de cumplir ciertas condiciones uno podra avnzar al siguiente mundo donde podra elegir el mundo al cual avanzar a continuacion teniendo para elegir entre montaña, campo nevado, cavernas o desierto.
